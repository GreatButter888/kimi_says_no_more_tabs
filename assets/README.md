# Assets Directory

## Icon Requirements

Place your extension icon as `icon.png` in this directory.

**Icon specifications:**
- Size: 128x128 pixels (will be scaled automatically for 16x16 and 48x48)
- Format: PNG with transparency
- Design: Should represent Kimi Räikkönen or the "NO" theme
- Suggestions:
  - Ice/snowflake icon
  - Kimi's helmet design
  - Simple "NO" text on colored background
  - F1 steering wheel

## Quick Icon Creation

You can create a simple icon using online tools:
- [Canva](https://www.canva.com) - Free design tool
- [Photopea](https://www.photopea.com) - Free Photoshop alternative
- [GIMP](https://www.gimp.org) - Free image editor

Or generate one using AI:
- Use DALL-E, Midjourney, or similar to generate: "minimalist F1 ice racing icon, Kimi Räikkönen inspired, 128x128"

## Placeholder Icon

Until you add your custom icon, Chrome will use a default extension icon.
